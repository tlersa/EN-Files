#### المميزات :
- مجانية ومفتوحة المصدر ✔️
- في حال لم تكن مثبت المكاتب المطلوبة (marshal - pyfiglet) سيتم تثبيتها تلقائياً ✔️
- تحفظ لك اسم الملف المشفر الجديد بكلمة New_ ثم إسم الملف الذي تم تشفيره✔️

#### تثبيت المكاتب
```
pip install marshal
pip install pyfiglet
```
